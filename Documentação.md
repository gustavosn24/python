# Biblioteca cennter
Projeto desenvolvido com python para gerenciamento de uma biblioteca

## Objetivo
 Criar um programa no terminal que permita gerenciar livros de uma pequena biblioteca.

## Funcionalidades do Sistema
  1 - Cadastrar livro
  2 - Listar livros
  3 - Emprestar livro
  4 - Devolver livro
  5 - Buscar livro
